# Validación de datos sintéticos para modelos de regresión

Este repositorio contiene el código utilizado en el Trabajo Final de Máster titulado 
"Validación de datos sintéticos para modelos de regresión: un estudio de simulación estadística".

## Contenido

- `R/`: funciones auxiliares utilizadas en el estudio.
- `scripts/`: scripts para reproducir cada caso de simulación.
- `TFM_simulacion_principal.Rmd`: archivo principal de análisis.
- `resultados/`: tablas y figuras generadas a partir de las simulaciones.

## Requisitos

El análisis fue desarrollado en R. Los principales paquetes utilizados son:

- `synthpop`
- `MASS`
- `dplyr`
- `ggplot2`
- `broom`
- `car`
- `tidyr`
- `purrr`

## Reproducibilidad

Para reproducir los resultados, ejecutar primero los scripts de la carpeta `R/` y posteriormente los scripts correspondientes a cada caso de simulación.
